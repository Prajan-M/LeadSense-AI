# LeadSense AI

LeadSense AI is a smart lead intelligence system that helps businesses identify high-value decision-makers such as CTOs and VP Engineering.

## Problem
Businesses struggle to identify the right leads and often rely on generic outreach, resulting in low engagement.

## Solution
Our system scores leads based on company size, role, and tech fit, and generates personalized outreach messages with A/B testing.

## Features
- Lead scoring (Hot / Warm / Cold)
- Decision-maker targeting (CTO, VP)
- Personalized outreach generation
- A/B testing for optimization

## User Flow
1. View leads on dashboard
2. Leads are scored and categorized
3. Select a lead to view details
4. Generate outreach messages (A/B)
5. Compare and choose best message
